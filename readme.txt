A comprehensive school tour booking system for managing parent visits and campus tours.

== Description ==

Campus Visit Scheduler is a powerful WordPress plugin designed specifically for schools to manage campus tour bookings. It allows parents and guardians to easily book school tours online, while giving administrators full control over tour schedules, capacity, and notifications.

= Key Features =

**For Parents:**
* Easy-to-use booking form with date and time selection
* Real-time availability display
* Instant email confirmation
* Add tour to calendar (ICS download)
* Mobile-responsive design

**For Administrators:**
* Configure recurring weekly tours or one-off special events
* Set capacity limits per time slot
* Manage blackout dates (holidays, closures)
* View and manage all bookings
* Calendar view for quick overview
* Export bookings to CSV
* Customizable email templates
* Multiple notification recipients
* Detailed reporting and statistics

= Shortcodes =

* `[campus_visit_scheduler]` - Display the booking form
* `[campus_visit_calendar]` - Display a tour availability calendar

= Getting Started =

1. Install and activate the plugin
2. Go to Campus Visits > Settings to configure your tour schedule
3. Add time slots for recurring tours or one-off events
4. Set your capacity and booking window preferences
5. Add the `[campus_visit_scheduler]` shortcode to any page
6. Start receiving bookings!

== Installation ==

1. Upload the `campus-visit-scheduler` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Campus Visits > Settings to configure the plugin
4. Add the shortcode `[campus_visit_scheduler]` to any page where you want the booking form to appear

== Frequently Asked Questions ==

= How do I set up tour times? =

Go to Campus Visits > Settings > Tour Schedule. You can add recurring tours (repeat weekly) or one-off tours for specific dates.

= Can I limit the number of bookings per time slot? =

Yes! When adding a time slot, you can specify the maximum number of booking groups allowed.

= How do I prevent bookings on holidays? =

Go to Campus Visits > Settings > Blackout Dates and add any dates when tours are unavailable.

= Can multiple staff receive booking notifications? =

Yes, go to Campus Visits > Settings > Notifications to add multiple email recipients.

= How do I customize the confirmation emails? =

Go to Campus Visits > Settings > Email Templates to customize all email content using placeholders.

= Is the plugin mobile-friendly? =

Yes, the booking form is fully responsive and works on all devices.
